# SWE-CI: Evaluating Agent Capabilities in Maintaining Codebases via Continuous Integration

## 概要
LLMエージェントの静的バグ修正を超えた、複雑な要件変更や長期的な機能イテレーションにおけるコードベース保守能力を評価するSWE-CIベンチマークを提案。実世界のソフトウェア開発における継続的インテグレーションパイプラインを模倣し、既存の一発修復パラダイムの限界を超えたエージェント評価を実現。
