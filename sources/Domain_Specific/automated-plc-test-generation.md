# Automated Control Logic Test Case Generation using Large Language Models

## 概要
産業オートメーションにおけるPLC・DCS制御ロジックのテストケース自動生成にLLMを適用。プロンプトにコードを提供してテストケースを合成させる新手法を提案。OSCATオートメーションライブラリの10個のオープンソース関数ブロックで実験し、低〜中複雑度プログラムで高い文カバレッジを達成。ただしLLM生成テストケースのアサーションに多くの誤りがあり手動修正が必要。
