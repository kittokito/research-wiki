# OpenClaw-RL: Train Any Agent Simply by Talking

## 概要
エージェントインタラクションが生成するnext-state信号（ユーザー返答、ツール出力、ターミナルやGUI状態変化）を即時のオンライン学習ソースとして活用するOpenClaw-RLフレームワークを提案。next-state信号は普遍的であり、ポリシーはそれらすべてから同時に学習可能という観察に基づく。
