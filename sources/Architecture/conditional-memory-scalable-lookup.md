# Conditional Memory via Scalable Lookup: A New Axis of Sparsity for Large Language Models

## 概要
MoEに対する補完的なスパース性メカニズムとして「条件付きメモリ」を導入。古典的N-gram埋め込みをO(1)ルックアップで近代化したEngramモジュールを提案し、ニューラル計算（MoE）と静的メモリ（Engram）のトレードオフを最適化するU字型スケーリング則を発見。Engramを27Bパラメータまでスケールし、知識検索・推論・コード・数学タスクで性能向上を実証。
