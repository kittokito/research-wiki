# Embarrassingly Simple Self-Distillation Improves Code Generation

## 概要
検証器・教師モデル・強化学習なしに、LLM自身の生の出力のみでコード生成を改善できることを実証。Simple Self-Distillation（SSD）として、特定の温度とトランケーション設定でモデルからソリューションをサンプリングし、標準的なSFTでファインチューニングするだけの極めてシンプルな手法を提案。
