# The Reversal Curse: LLMs trained on 'A is B' fail to learn 'B is A'

## 概要
自己回帰LLMの驚くべき汎化の失敗を明らかにした論文。モデルが「AはBである」形式の文で学習した場合、逆方向の「BはAである」に自動的に汎化しないことを実証。これを「Reversal Curse（逆転の呪い）」と命名。

## メモ
cs.CL, cs.AI, cs.LG。2023年9月公開。LLMの根本的な制約に関する重要な発見。
