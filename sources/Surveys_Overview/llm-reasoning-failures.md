# Large Language Model Reasoning Failures

## 概要
LLMの推論失敗に関する初の包括的サーベイ。推論を embodied / non-embodied に分類し、non-embodied をさらに informal / formal に細分化。失敗を「基本的アーキテクチャ限界」「アプリケーション固有の制約」「ロバストネスの不整合」の3タイプに分類し、各失敗の根本原因と緩和策を分析している。

- GitHubリポジトリで関連研究を整理・公開している
