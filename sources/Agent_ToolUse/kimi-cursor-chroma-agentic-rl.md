# How Kimi, Cursor, and Chroma Train Agentic Models with RL

## 概要
エージェントAIモデルを強化学習で訓練する3つの技術的アプローチを分析。Moonshot AIのKimi K2.5（並列エージェントオーケストレーション）、CursorのComposer 2（自己要約と本番データからのリアルタイムRL）、ChromaのContext-1（文書プルーニングによる能動的コンテキスト管理の学習）。3システムとも本番環境での学習、成果ベースの報酬、大規模非同期ロールアウトという共通方法論を共有。
