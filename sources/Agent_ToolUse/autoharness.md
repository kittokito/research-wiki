# AutoHarness: improving LLM agents by automatically synthesizing a code harness

## 概要
Gemini-2.5-Flashが反復的改良を通じてLLMの周りに保護的なコード構造（ハーネス）を自動合成できることを実証。145のTextArenaゲームで不正な行動を防止。小さなモデルがGemini-2.5-Proなどの大きなモデルを凌駕可能に。さらにランタイムLLM呼び出しを排除する完全なアクションポリシーのコード生成を実現し、コストを大幅に削減。
