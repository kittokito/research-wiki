# Drop the Hierarchy and Roles: How Self-Organizing LLM Agents Outperform Designed Structures

## 概要
25,000タスクの計算実験で8モデル・4〜256エージェント・8つの協調プロトコルを検証。エージェントが事前割り当てなしに自発的に専門役割と浅い階層構造を発展させることを実証。逐次ハイブリッドプロトコルが中央集権的協調を14%上回り、256エージェントまでスケーリング可能。オープンソースモデルがクローズドソースの95%の性能を大幅に低いコストで達成。
