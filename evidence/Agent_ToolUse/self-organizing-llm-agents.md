# Self-Organizing LLM Agents からの抽出

## 主要な主張
- エージェントが事前割り当てなしに自発的に専門役割と浅い階層構造を発展 [source](../../sources/Agent_ToolUse/self-organizing-llm-agents.md)
- 逐次ハイブリッドプロトコルが中央集権的協調を14%上回る [source](../../sources/Agent_ToolUse/self-organizing-llm-agents.md)
- 256エージェントまでスケーリング可能で品質維持 [source](../../sources/Agent_ToolUse/self-organizing-llm-agents.md)
- オープンソースモデルがクローズドソースの95%の性能を大幅に低いコストで達成 [source](../../sources/Agent_ToolUse/self-organizing-llm-agents.md)
